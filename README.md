# Product Card UI

A responsive product card component designed for e-commerce and product-based websites.

## Features
- Clean and modern UI
- Product image with details
- Price and call-to-action button
- Responsive design (mobile & desktop)
- Reusable card component

## Tech Stack
- HTML5
- CSS3
## What I Learned
- Card layout using Flexbox/Grid
- Spacing, alignment, and typography
- Creating reusable UI components
- Designing for responsiveness
