# Product Card UI

A responsive product card component designed for e-commerce websites to showcase product details like image, price, and action button.

## Overview
This project demonstrates a visually appealing product card UI built with HTML and CSS. It highlights clean layout, responsive design, and reusable structure suitable for e-commerce sites.

## Features
- Clean and modern design  
- Displays product image, title, price, and CTA  
- Fully responsive layout for mobile and desktop  
- Reusable component structure

## Tech Stack
- HTML5  
- CSS3
